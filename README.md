📊 AWT Sales Manager — ERP & Gestão de Vendas Profissional








🚀 Sobre o Projeto

O AWT Sales Manager é um sistema ERP moderno desenvolvido para gestão de vendas e equipes, com foco em:

📊 Controle operacional completo
🔐 Segurança robusta
⚡ Alta performance
📱 Interface responsiva e intuitiva

Inspirado em sistemas reais de mercado, o projeto entrega uma base sólida para aplicações comerciais escaláveis.

⚙️ Instalação
1. Clone o repositório
git clone https://github.com/awaldige/awt-sales-manager.git
2. Crie o banco de dados
CREATE DATABASE awt_sales;
3. Importe o schema
/database/schema.sql
4. Configure o sistema

Edite:

/includes/config.php
$host = "localhost";
$user = "root";
$pass = "";
$db   = "awt_sales";
